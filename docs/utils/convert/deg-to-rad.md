Degrees to radians
==================

### `degToRad`

Method converts angle in degrees to radians.


Usage
-----

```js
const radians = MathUtils.convert.degToRad(degrees);
```


### Arguments

* **`degrees`**: *`number`* - angle in degrees.


### Returns

* *`number`* - angle in radians.


Examples
--------

```js
MathUtils.convert.radToDeg(180) === Math.PI;
```
