Degrees to radians
==================

Method converts angle in degrees into radians.


Usage
-----

```js
const radians = MathUtils.convert.degToRad(degrees);
```


### Arguments

* `<number>` `degrees` - angle in degrees.


### Returns

* `<number>` - angle in radians.


Examples
--------

```js
MathUtils.convert.radToDeg(180) === Math.PI;
```
