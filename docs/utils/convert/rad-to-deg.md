Radians to degrees
==================

### `radToDeg`

Method converts angle in radians into degrees.


Usage
-----

```js
const degrees = MathUtils.convert.radToDeg(radians);
```


### Arguments

* **`radians`**: *`number`* - angle in radians.


### Returns

* *`number`* - angle in degrees.


Examples
--------

```js
MathUtils.convert.radToDeg(Math.PI) === 180;
```
