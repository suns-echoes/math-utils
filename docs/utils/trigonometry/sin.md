Trigonometry / Sine
===================

### `sin`

Method calculates sine value.


Usage
-----

```js
const value = MathUtils.trigonometry.sin(theta);
```


### Arguments

* **`theta`**: *`number`* - angle in radians


### Returns

* *`number`* - sine value


Examples
--------

```js
// Calculate value:
const value = MathUtils.trigonometry.sin(1.53);
```
