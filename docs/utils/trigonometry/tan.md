Trigonometry / Tangent
======================

### `tan`

Method calculates tangent value.


Usage
-----

```js
const value = MathUtils.trigonometry.tan(theta);
```


### Arguments

* **`theta`**: *`number`* - angle in radians


### Returns

* *`number`* - tangent value


Examples
--------

```js
// Calculate value:
const value = MathUtils.trigonometry.tan(1.53);
```
