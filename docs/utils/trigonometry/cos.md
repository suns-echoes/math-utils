Trigonometry / Cosine
=====================

### `cos`

Method calculates cosine value.


Usage
-----

```js
const value = MathUtils.trigonometry.cos(theta);
```


### Arguments

* **`theta`**: *`number`* - angle in radians


### Returns

* *`number`* - cosine value


Examples
--------

```js
// Calculate value:
const value = MathUtils.trigonometry.cos(1.53);
```
